Meu Banco

O Meu banco é aplicativo preparado para receber dados de extrato e fatura do cartão de crédito. Leve, rápido e acessível.

Para quem?

O Meu banco é um aplicativo simples desenvolvido para um treinamento e disponibilizado para referência do mesmo.

Páginas

  • Login
  • Extrato
  • Fatura do Cartão de Crédito
  • Perfil do Cliente

Tecnologias utilizadas

  • CORDOVA
  • IONIC
  • SASS
  • ANGULARJS
  • HTML
  • CSS
  • JAVASCRIPT
  • GULP
